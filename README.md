# An-Agent-based-and-Equation-based-Meta-Model-of-Behavioral-Sink-the-Case-of-Universe-25
A minimal dual-model framework to study population collapse dynamics driven by density–stress feedbacks, combining a deterministic equation-based model (EBM) and a stochastic agent-based model (ABM), calibrated on empirical data (Universe 25) and evaluated via modal-cluster analysis.
